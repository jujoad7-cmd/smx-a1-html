Fin del soporte de Windows 10

Microsoft dejó de actualizar Windows 10, dejando millones de PCs sin soporte y provocando un aumento en las ventas de equipos nuevos con Windows 11.

Los equipos que sigan usando Windows 10 quedan más expuestos a virus, ciberataques y vulnerabilidades.

Microsoft ofrece una opción de soporte extendido de pago (ESU) para empresas, pero es temporal y costoso.

Autor: Adrian Bergés - 16/10/2025
